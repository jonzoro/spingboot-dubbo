# SpringBoot整合Dubbo项目
这个项目主要是用来测试部署SpringBoot和Dubbo框架。

## 环境准备
- IDEA/JDK/GIT
- zookeeper
- dubbo-admin

## 项目运行
- git clone 本项目
- 启动zookeeper
- 启动dubbo-admin
- 启动dubbo-provider
- 启动dubbo-customer
